# Animal Detection Model 🐾

A simple application to detect animals in images and videos using a custom-trained YOLOv8 model. Carnivorous animals are highlighted and counted with a pop-up alert.

## 🔍 Features
- Detect multiple animals per frame
- Highlights carnivores: **cat, dog, bear**
- Supports both **images and videos**
- GUI built with **Tkinter**
